-# Implementando Kanban

Nesta parte, transformaremos nosso aplicativo de Notas em um aplicativo Kanban. Durante o processo, você aprenderá os conceitos básicos de React. Como React é apenas uma biblioteca de UI e também discutiremos tecnologias para gerir o estado da sua aplicação. Vamos configurar uma solução de gerenciamento de estado para nossa aplicação. Também veremos como usar React DnD para adicionar funcionalidades de arrastar e soltar no nosso quadro Kanban.
