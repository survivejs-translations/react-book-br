# Estilos de escrita

* *nome de arquivo*
* `código`

## npm

* *npm* deve ser escrito em letra minúscula, mesmo que seja no início de uma frase..

Uso recomendado:

```bash
npm i alt --save
```

Sempre que instalar algo com *npm*, lembre-se de incuir `--save` ou `--save-dev`!
