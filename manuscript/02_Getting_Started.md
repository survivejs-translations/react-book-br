-# Começando

React, apesar de ser uma biblioteca criada recentemente, teve um impacto significativo na comunidade de desenvolvimento front-end. Ele introduziu conceitos, como o **virtual DOM** e fez a comunidade entender o poder dos componentes. Sua abordagem de design orientada a componentes funciona bem para a web. Mas React não se limita à web. Você pode usá-lo para desenvolver interfaces de usuário para aplicativos móveis.

Nesta parte, começaremos o mão na massa em React e implementar uma pequena aplicação de Notas. Isso é algo que eventualmente se tornará um quadro Kanban. Você aprenderá o básico de React e irá se acostumar com seu fluxo de trabalho.
